# Weather Dashboard

A real-time weather application with React frontend and Node.js backend.

## Features
- Live weather data from OpenWeatherMap API
- Clean, responsive UI
- Error handling


 Set up backend
   ```bash
   cd server
   npm install
   echo "OPENWEATHER_API_KEY=your_key" > .env
   node server.js
   ```
 Set up frontend
   ```bash
   cd ../client
   npm install
   npm start
   ```
